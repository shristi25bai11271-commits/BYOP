# BYOP
BookMatch AI is a Python-based application that recommends books based on user input and enhances results with AI-generated descriptions. It uses the Open Library API to provide real-time, personalized suggestions.
📚 BOOKMATCH AI
This project is about creating a simple Python program that suggests books based on a user’s personal preferences.
Many students and readers struggle to find books that match their interests. This is a common problem among college students, including my peers, who often spend a lot of time searching but are unsure what to choose. Many people rely on random suggestions from social media or friends, which may not always suit their taste.
To solve this, the program first asks the user a few questions, such as their preferred genre (like action, thriller, or romance), the type of book length (single book, trilogy, or series), and the kind of protagonist they like. These inputs help the system understand the user’s reading preferences.
After collecting the responses, the program uses them to search for books through the Open Library API, which contains a large collection of books. It then selects the top three books that best match the user’s choices.
To make the results more engaging, the project also uses AI tools like the **OpenAI API to generate short descriptions and review-style summaries for each book.
Finally, the program displays the recommended books along with their titles, authors, cover image links, and a link where users can explore more details.
